# Soldiers-Monogame
Monogame implementation of Commandos 1 game series written in C#, does requiere to have original game files.

# Prerequisites

    -Windows
    MonoGame: 3.4 or higher
    OpenTK 1.1
    
    -Linux
    OpenGL 3.0
    SDL2
